# Bank Demo - Microservices Architecture

## Features
- Microservices (Order, Payment, Notification)
- Event-driven architecture
- Idempotency (processed_events)
- Circuit Breaker (Payment Service)
- Logging (logs table)
- API Gateway
- Frontend (EJS)

## Run

npm install

# start services
npm run start:order
npm run start:payment
npm run start:notification
npm run start:gateway
npm run start:frontend
